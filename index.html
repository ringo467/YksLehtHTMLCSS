Sissejuhatus

HTML tähendab HyperText Markup Language. See on keel, millega luuakse ja kirjeldatakse veebilehtede struktuuri.

Lihtsamalt öeldes: HTML ütleb veebilehitsejale, millised elemendid lehel on (nt pealkirjad, lõigud, lingid, pildid, tabelid). See ei määra otseselt, kuidas need välja näevad – selleks kasutatakse tavaliselt CSS-i (stiilid) ja dünaamiliseks käitumiseks JavaScripti.

Teooria

HTML ehk hypertext markup language on veebilehtede struktuuri kirjeldamise keel see ei ole programmeerimiskeel vaid märgistuskeel millega määratakse millised osad lehel olemas on näiteks pealkirjad lõigud pildid või lingid

Iga html dokument algab doctype deklaratsiooniga ja koosneb html elemendist mille sees on kaks osa head ja body head sisaldab metaandmeid lehe kohta näiteks pealkiri või kasutatav tähemärkide kooditabel body sisaldab tegelikku sisu mida kasutaja ekraanil näeb

HTML koosneb elementidest mida tähistavad sildid avav ja sulgev silt nende vahel on elemendi sisu mõned elemendid on plokk elemendid mis algavad alati uuelt realt näiteks div p või h1 teised on reaelementid mis jäävad samale reale näiteks span a või strong

Peamised html elemendid on pealkirjad h1 kuni h6 lõigud p lingid a pildid img loendid ul ja ol tabelid table tr td ja vormid form input button

Elementidele saab lisada atribuute mis annavad lisainfot näiteks a href määrab lingi aadressi ja target blank avab selle uues aknas

HTML5 tõi juurde semantilised elemendid mis teevad koodi loetavamaks ja aitavad otsingumootoreid ning ekraanilugejaid näiteks header nav main section article ja footer

Kokkuvõttes on HTML veebilehe alus CSS annab sellele välimuse ja javascript dünaamika
